Recommended setup on Windows:

1. Create a virtual environment and install dependencies
   conda create -n ai-radar python=3.11 -y
   D:\anaconda\envs\ai-radar\python.exe -m pip install -r requirements.txt

2. Register the scheduled tasks from the repo root
   powershell -ExecutionPolicy Bypass -File .\scripts\register_tasks.ps1 -PythonExe "D:\anaconda\envs\ai-radar\python.exe"

3. Test the daily push manually before tomorrow 08:00
   powershell -ExecutionPolicy Bypass -File .\scripts\run_pipeline.ps1 -PythonExe "D:\anaconda\envs\ai-radar\python.exe" -Task daily -Deliver

Tasks created by `scripts/register_tasks.ps1`:

1. AI Radar Ingest
   Trigger: Daily at 07:00
   Command: powershell.exe -NoProfile -ExecutionPolicy Bypass -File ".\scripts\run_pipeline.ps1" -PythonExe "D:\anaconda\envs\ai-radar\python.exe" -Task ingest

2. AI Radar Tier1
   Trigger: Daily at 07:30
   Command: powershell.exe -NoProfile -ExecutionPolicy Bypass -File ".\scripts\run_pipeline.ps1" -PythonExe "D:\anaconda\envs\ai-radar\python.exe" -Task tier1

3. AI Radar Daily Curate
   Trigger: Daily at 07:50
   Command: powershell.exe -NoProfile -ExecutionPolicy Bypass -File ".\scripts\run_pipeline.ps1" -PythonExe "D:\anaconda\envs\ai-radar\python.exe" -Task daily-curate

4. AI Radar Daily Digest
   Trigger: Daily at 08:00
   Command: powershell.exe -NoProfile -ExecutionPolicy Bypass -File ".\scripts\run_pipeline.ps1" -PythonExe "D:\anaconda\envs\ai-radar\python.exe" -Task daily -Deliver

5. AI Radar Tier2
   Trigger: Weekly Sunday at 11:00
   Command: powershell.exe -NoProfile -ExecutionPolicy Bypass -File ".\scripts\run_pipeline.ps1" -PythonExe "D:\anaconda\envs\ai-radar\python.exe" -Task tier2

6. AI Radar Weekly Digest
   Trigger: Weekly Sunday at 12:00
   Command: powershell.exe -NoProfile -ExecutionPolicy Bypass -File ".\scripts\run_pipeline.ps1" -PythonExe "D:\anaconda\envs\ai-radar\python.exe" -Task weekly -Deliver

Logs are written to `state/logs/`.
